This is a Microservices-based application built using Java and Spring Boot. It demonstrates key architectural principles such as service separation, inter-service communication, and containerization. 
The project includes multiple microservices, each responsible for a specific domain, communicating via REST APIs or messaging queues.
The goal of this project is to simulate a real-world scalable system with best practices including Docker, CI/CD, and configurable environments.

Tech Stack
Java 23

Spring Boot

Spring Cloud (Config, Discovery/Eureka, Gateway)

REST APIs

MySQL/PostgreSQL

Docker & Docker Compose

Git & GitHub Actions (for CI/CD)
